# 启动指南

## 环境要求

- Node.js >=20.18.0 <25.0.0
- pnpm >=8.0.0

## 安装依赖

```bash
pnpm install
```

## 开发模式（热更新）

同时启动 server 和 client：

```bash
pnpm run dev
```

- Client: http://localhost:5173
- Server: http://localhost:3001

## 生产模式

```bash
# 1. 构建
pnpm run build

# 2. 启动 server
node server/dist/index.js
```

## PM2 生产部署

在项目根目录创建 `ecosystem.config.cjs`：

```js
module.exports = {
  apps: [{
    name: 'modelhub-server',
    script: 'server/dist/index.js',
    env: {
      NODE_ENV: 'production',
      PORT: 3001,
    },
    instances: 1,
    exec_mode: 'fork',
  }]
}
```

然后运行：

```bash
pnpm run build
pm2 start ecosystem.config.cjs
pm2 save
pm2 startup
```

> `instances` 必须为 1，因为 SQLite 不支持多进程并发。

## 数据存储

- 数据库位置：`server/data/freeapi.db`（SQLite）
- 加密密钥：`server/data/.encryption-key`
- 自定义路径：设置环境变量 `FREEAPI_DB_PATH`

迁移数据只需复制 `server/data/` 目录下所有文件到新环境。
